# Planejamentos

Registro dos planos produzidos pelos agentes **plan-ops** e **plan-dev** antes de
serem executados por **run-ops** / **run-dev**.

Convenção de nomes:

- `AAAA-MM-DD-ops-<assunto>.md` — planos de infraestrutura (plan-ops)
- `AAAA-MM-DD-dev-<assunto>.md` — especificações de arquitetura/testes (plan-dev)

Atualize o arquivo existente quando o plano for revisado, em vez de duplicá-lo.
